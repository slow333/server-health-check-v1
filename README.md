## 서버 정기점검 자동화
## 서버 헬스체크
## 서버 설정 자동화

## 환경 구성
> 사용 언어: python version 13.9

> 사용 라이브러리: flask, Flask-login, Flask-sqlalchemy, etc

> 필요사항

- 사용자가 웹에서 서버 접속
  - 권한 관리 필요
- 프로그램에서 서버에 원격 접근
  - 권한에 따라 접속 가능 서버 제한
- 서버에서 데이터 가져오기
  - Flask 앱이 SSH로 서버에 접속 
  - 서버 상태 정보 수집 
- 가져온 데이터를 DB에 저장
- 저장된 데이터를 웹에 출력하기
  - 웹에서 시각화
